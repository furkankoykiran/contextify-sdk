# Getting help

## Documentation

- [README.md](README.md) — what this package is and how to install (once a
  release exists).
- Upstream project: [Contextify](https://contextify.live).

## Channels

- **Bugs and feature requests:** open a GitHub issue using a template.
- **Security:** see [`SECURITY.md`](SECURITY.md).
- **Conduct:** see [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md).
- **SaaS account / billing / dashboard:** send to **support@contextify.live**.

## Scope of this repository

This is the public OSS package for `contextify-sdk`. Issues belong here when
they concern the package's behavior, install, API, or examples. Anything
about the hosted Contextify service goes to support@contextify.live.
