# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be respectful. Disagreement is fine; harassment is not.
- Assume good faith. Ask before assuming malice.
- Critique the work, not the person.
- Confidential context stays in private channels.

## Enforcement

Reports go to **conduct@contextify.live**. Maintainers will acknowledge a
report within 3 business days. Maintainers who violate this code receive the
same enforcement as any other contributor.
